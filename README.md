# TubePlay Premium

Ad-free YouTube player with background play support.

## Features
- 🔍 Search YouTube
- 🎵 Audio-only mode (background play)
- 📺 Video mode
- 🚫 No ads
- 🕒 Watch history
- 📱 Mobile friendly

## Usage
1. Search for any YouTube video
2. Select Audio or Video mode
3. Click on a video to play
4. Enjoy ad-free playback!
